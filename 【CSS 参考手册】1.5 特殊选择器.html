1.5 特殊选择器

  • 伪类

    • ::frist-line

伪类

::frist-letter

  • 首字
p::first-letter {
    font-size: 24px;
    color: green;
}


<p>创新者和应用程序开发人员在这个新的区块链经济中面临的一个挑战是从头构建新的区块链应用程序是很困难的。最重要的是，通过传统的工作量证明（PoW）和权益证明机制（PoS），网络和应用程序的安全性取决于计算能力和/或 token。对于小企业主和创业公司来说，这些挑战使得门槛很高。小型创业公司无法独立部署广泛分布而功能强大的计算机网络来保护其应用程序。</p>
